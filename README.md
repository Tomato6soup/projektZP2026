**Info o projekcie**


**Projekt został wykonany z wykorzystaniem następujących technologii:**

C#, WPF (Windows Presentation Foundation), MVVM (Model–View–ViewModel), SQL Server, SQL Server Management Studio (SSMS), Visual Studio
GitHub - dla monitorowania pracy zespołowej.

** Główne funkcjonalności**

✔ logowanie użytkowników
✔ obsługa ról użytkowników
✔ wyszukiwanie publikacji po tytule
✔ filtrowanie publikacji
✔ przeglądanie projektów
✔ wyświetlanie szczegółów projektów
✔ wyświetlanie polubionych projektów
✔ dodawanie publikacji (administrator)
✔ tryb jasny i ciemny


Projekt zawiera bazę danych zrobioną za pomocą SQL Management Studio, posiada wpisane przykładowe dane.

Mamy okieńko dla logowania użytkownika do systemu. Są trzy rolę: Student, Pracownik, Administrator. Powitanie na stronie głównej zależy od wprowadzonej roli przez użytkownika.

Można zmienić na tryb ciemny/jasny.
**Są 4 zakładki:** Strona Główna, Projekty, Publikacje oraz Konferencje.
W wyszukiwarce można szukać prace naukowę za nazwą tytułową.
W "Projketach" za pomocą przycisku "Szczegóły" widać opis projektu.
Na stronie głównej są wyświetlane polubione projekty.
Także jest opcja filtrowania wg roku publikacji, typu oraz wydawnictwa.
Opcja "Dodaj publikację" jest dostępna tylko i wyłącznie dla Administratora.
